🚀 快速部署指南 | QUICK DEPLOYMENT GUIDE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  推送到 GitHub
   git remote add origin https://github.com/YOUR_USERNAME/tcml-web.git
   git push -u origin main

2️⃣  在 Vercel 連接
   訪問: https://vercel.com/new
   選擇 GitHub 倉庫：tcml-web
   自動檢測 Next.js 設定

3️⃣  配置環境變數（可選）
   NEXT_PUBLIC_SITE_URL: https://www.tcml-mandarin.org

4️⃣  部署
   點擊 "Deploy" 按鈕
   等待 2-3 分鐘完成
